- `np.dot(weights, V)` 的本质是坐标重绘。它不是在创造新词，而是根据上下文的引力（Weights），将原始词向量从孤立的坐标点，拉向它在当前语境中该有的新位置。这个‘位移’的过程，就是模型产生‘理解’的过程。
- 为了防止指数爆炸，对于softmax函数我们引入常数 $C = \max(x)$ 利用平移不变性进行计算：

$$\text{Softmax}(x_i) = \frac{\exp(x_i - \max(x))}{\sum_{j} \exp(x_j - \max(x))}$$
